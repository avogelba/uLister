1. О программе
  • uLister это мощное средство просмотра которое позволяет просматривать более 500 различных форматов файлов.
  • Text Search позволяет искать файлы, содержащие заданный текст в документах поддерживаемых типов и вложениях всех уровней.
  • Document Converter позволяет конвертировать просматриваемые файлы в txt,pdf,html,mhtml, jpeg 2000, jpeg, tiff, png, gif, bmp.
2. Поддерживаемые форматы
Множество популярных форматов текстовых процессоров, электронных таблиц, презентаций, растровых и векторных изображений.
Полный список:

5. Настройка
Параметры хранятся в ulister.ini поиск которого производится в следующем порядке:
  1. каталог плагина
  2. %COMMANDER_INI%
  3. %APPDATA%
Если вы хотите редактировать настройки из плагина убедитесь что ulister.ini доступен для записи.
Если  ulister.ini не найден то он будет создан в %APPDATA%
    • Параметр noloadtypes содержит список запрещенных к загрузке форматов.
    • Параметр onlyloadtypes содержит список разрешенных к загрузке форматов.
    • Параметр nopreviewtypes содержит список запрещенных к показу эскизов форматов.
    • Параметр onlypreviewtypes содержит список разрешенных показу эскизов форматов.
    • Параметр nosearchtypes содержит список запрещенных форматов для поиска файлов, содержащих заданный текст.
    • Параметр onlysearchtypes содержит список разрешенных форматов для поиска файлов, содержащих заданный текст.
    • Параметр keepinmemory отвечает за возможность не выгружать библиотеки Outside In Viewer из памяти после первого использования,что уменьшает время загрузки документа, но увеличивает потребляемую память (По умолчанию разрешено).
    • Параметр optionsdir содержит путь к каталогу в котором будут сохраняться настройки (Поддерживаются переменные окружения).
    • Параметр showtoolbar отвечает за панели инструментов (По умолчанию разрешено).
6. Установка
Все перечисленные ниже файлы можно загрузить со страницы http://www.oracle.com/technetwork/middleware/content-management/downloads/oit-dl-otn-097435.html, для этого необходимо принять лицензию и зарегистрироваться на сайте.
Lister плагин:
Установите плагин из архива.
Загрузите vw-8-3-5-win-x86-32.zip : Viewer Technology 8.3.5 Windows (x86-32)
Скопируйте каталог redist в каталог плагина.
Content плагин:
Добавьте ulister.wlx, как wdx плагин в TotalCommander.
Загрузите sx-8-3-5-win-x86-32.zip : Search Export 8.3.5  Windows (x86-32)
Скопируйте каталог redist в каталог плагина, без замены файлов.
Document Converter:
Если Вы хотите конвертировать несколько файлов за раз, Вам необходимо установить плагин как wcx, для этого добавьте в wincmd.ini:
[PackerPlugins]
convert=23,path to uLister\ulister.wlx
text:
Загрузите sx-8-3-5-win-x86-32.zip : Search Export 8.3.5 Windows (x86-32)
Скопируйте каталог redist в каталог плагина, без замены файлов.
pdf:
Загрузите px-8-3-5-win-x86-32.zip : PDF Export 8.3.5 Windows (x86-32)
Скопируйте каталог redist в каталог плагина, без замены файлов.
html, mhtml:
Загрузите hx-8-3-5-win-x86-32.zip : HTML Export 8.3.5 Windows (x86-32)
Скопируйте каталог redist в каталог плагина, без замены файлов.
jpeg 2000, jpeg, tiff, png, gif, bmp:
Загрузите ix-8-3-5-win-x86-32.zip : Image Export 8.3.5 Windows (x86-32)
Скопируйте каталог redist в каталог плагина, без замены файлов.
Обозреватель закладок:
Для просмотра списка файлов для которых есть закладки, добавьте ulister.wlx, как wfx плагин в TotalCommander.
7. Сочетания клавиш
  • Ctrl M - Сохранить текущую позицию
  • Ctrl T - Показать/Убрать панель инструментов
  • Ctrl O - Опции
  • Ctrl Z - Восстановить сохраненную позицию
  • Ctrl S - Диалог конвертирования
  • Ctrl + - Увеличить масштаб
  • Ctrl - - Уменьшить масштаб
